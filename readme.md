# Readme
Code suite for the replication package of “[Tariffs on Medical Goods: Pass-through, Geography, and Aggregate Costs to the US Healthcare System](https://katherynruss.weebly.com/uploads/4/5/1/7/45179109/tariffs_on_medical_goods_wp.pdf)”
with Kang, Russ, and Waters (2025).

This repo includes only the replication code to compile the data. See the complete replication package [here](https://www.dropbox.com/scl/fi/mx97iyhbsovaa0gqf31x3/replication_package_wp.zip?rlkey=jc57vw86sh2gpru54y4tp3l5c&st=tss3okgu&dl=0), which includes the tariff table and the Stata code used for estimation.
